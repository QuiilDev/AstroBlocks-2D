# AstroBlocks-2D
My project in python(pygame) in .exe

You'll appear in a randomly generated 2D world.
Pressing Esc opens a menu with difficulty settings and the option to exit the game. Your inventory can be accessed by pressing the "i" key, but it doesn't have many blocks at the moment.
You can build a 4x5 stone frame, which activates a portal to another dimension—the moon. The moon is also completely randomly generated, with altered gravity and large craters.
You can build various mechanisms with green wires, pistons, and levers. 
There are also rails with minecarts, so you can build a convenient underground subway. 
Fight zombies with a bow! (Zombies appear on the hard difficulty level.) 
Store items in chests!
You can build anything in this game—a regular house, a tower, or even pixel art. Good luck!
